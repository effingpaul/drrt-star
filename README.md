# Mutli-Robot-MPNet

## Implementation of dRRT*
- Clone the repo and switch to project directory:
```
git clone https://github.com/urastogi885/multi-robot-mpnet.git
cd multi-robot-mpnet/
```
- Now, run the 3 algorithms mentioned the dRRT* paper:
```
python dRRT.py
python aodRRT.py
python dRRT_star.py
```
